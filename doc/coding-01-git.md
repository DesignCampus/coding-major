Gitってなに？

--

<div class="box">
様々なファイルを管理するための仕組みで、変更なども記録しておける<br>
ファイル自体は使用者のローカルにすべて保存される<br>
それらを複数人で共有するためのリモートリポジトリを使用して履歴迄が共有することができる
</div>

---

Githubってなに？

--

<div class="box">
Gitのリモートリポジトリを管理するツール  
プロジェクト管理などの機能もある
</div>
